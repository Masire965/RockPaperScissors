# Rock Paper Scissors CLI Game 🎮

A simple **command-line Rock Paper Scissors game** built using Python. This project allows a user to play against the computer in a classic game of chance.

---

## 📌 Features

* Play against the computer 🤖
* Random computer moves
* Simple CLI (Command Line Interface)
* Continuous gameplay option
* Beginner-friendly Python project

---

## 🛠️ Technologies Used

* Python 3

---

## 🚀 How to Run the Game

### 1. Clone the repository

```bash
git clone https://github.com/yourusername/RockPaperScissors.git
```

### 2. Navigate to the project folder

```bash
cd RockPaperScissors
```

### 3. Run the game

```bash
python RockPaperScissors.py
```

---

## 🎮 How to Play

1. Run the program
2. Choose one option:

   * Rock 🪨
   * Paper 📄
   * Scissors ✂️
3. The computer will randomly choose its move
4. The winner is displayed
5. Choose whether to play again

---

## 🧠 Game Rules

* Rock beats Scissors
* Scissors beats Paper
* Paper beats Rock
